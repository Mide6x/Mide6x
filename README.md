## What I’m building

### [Veritas Social](https://github.com/Veritas-Social)

A student-first social and career network designed to help students connect across campuses, discover opportunities, join verified communities, and communicate across language barriers.

Some parts I have worked on include:

- Cross-platform mobile development with **React Native and Expo**
- A production **Node.js and Express** API with Supabase and PostgreSQL
- Real-time messaging, typing indicators, reactions, voice notes, and Socket.IO events
- One-to-one WebRTC audio calls with TURN infrastructure
- Communities, feeds, events, notifications, and recommendation systems
- Redis caching, BullMQ background jobs, document processing, and worker pipelines
- Authentication hardening, risk controls, rate limiting, device checks, and **Aegis 2.0** security features
- AI-assisted recommendations, CV processing, and multilingual product experiences

> Most of the production source is private, so the public contribution graph only shows the contribution count. I’m happy to discuss the architecture, engineering decisions, and the systems I personally built.
